The chain config for all the networks supported by Router including the Gateway contract addresses can be found at the following links:

### For Mainnet

#### Gateway Contracts
<APIData
  apiData={[
    { contractConfigUrl: 'https://sentry.lcd.routerprotocol.com/router-protocol/router-chain/multichain/contract_config?pagination.limit=10000', 
    chainConfigUrl: 'https://sentry.lcd.routerprotocol.com/router-protocol/router-chain/multichain/chain_config', 
    networkType: 'Mainnet', contractType: 'GATEWAY' }
  ]}
/>

<div style={{ marginBottom: '50px' }}></div>

:::info
Please make sure that you pass the **Chain ID** and **Gateway Contract Address** in your request without any quotes. For example, for Polygon, instead of passing Chain ID as "137" or '80001371', pass it as 137.
:::
