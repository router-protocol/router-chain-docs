Introduction

The guides in this section will explain the process of using various tools to simplify building and maintaining dApps using Router.

Important Tools​

Router Explorer
A block explorer to monitor cross-chain transactions.
Router Faucet
A tool to get testnet tokens for performing tasks on Testnet.
Router Station
A tool to manage contract lifecycle-related processes on the Router chain.
Router Hub
An tool for delegating ROUTE tokens to validators.
Router Sandbox
An easy-to-use prototyping environment for interoperable dApps built using Router.
